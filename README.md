# Data Mining and Machine Learning Project

**Dataset:** [Most Streamed Spotify Songs 2023](https://www.kaggle.com/datasets/nelgiriyewithana/top-spotify-songs-2023)

[**Kaggle project**](https://www.kaggle.com/code/janastanisavoska/cpa-msss)

## Project Description

This is a collaborative project, from the second year, conducted by two Computer Science students as part of the Data Mining and Machine Learning course.

This project aims to investigate how the most-streamed songs of 2023 perform across different music streaming platforms. 

The dataset includes information such as: 
* track name, artist(s) name,
* release date,
* Spotify playlists and charts,
* streaming statistics,
* Apple Music playlists and charts,
* Deezer playlists and charts,
* Shazam charts,
* and various audio features (acousticness, speechiness, etc.).

The initial idea for this project is to start with Data Preprocessing, where we clean and preprocess the dataset to handle missing values, duplicates and inconsistencies, ensuring the data is accurate and reliable for further analysis. Following this, we will perform Exploratory Data Analysis (EDA) to examine the data and closely understand its patterns and connections. It serves as an initial step to more complex data analysis and modelling. After gaining insights from EDA, we will move on to Cross-platform Analysis, where we analyse the presence and performance of the songs across different platforms such as Spotify, Apple Music, Deezer, and Shazam. This involves comparing various metrics like the number of playlists a song is included in, its presence and rank on various charts etc.. This approach will provide valuable insights into the performance of songs across different streaming services. The final step is Data Modeling, where we develop predictive models using machine learning algorithms. These models will be tested using the preprocessed data, and their performance will be evaluated and optimized.
